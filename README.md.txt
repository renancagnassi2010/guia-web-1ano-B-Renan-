# Guia Profissional de Desenvolvimento Web

Um site educativo completo que explica os fundamentos do desenvolvimento web, boas práticas profissionais, panorama de tecnologias e o fluxo completo de um projeto web.

## Estrutura do Projeto
guia-web-1ano-renan
├── index.html # Página inicial com introdução a HTML, CSS e JS
├── tecnologias.html # Panorama de tecnologias web com filtros
├── boas-praticas.html # Boas práticas de desenvolvimento
├── fluxo.html # Fluxograma de um projeto web completo
├── quiz.html # Quiz de conhecimentos
├── css/
│ └── styles.css # Estilos principais do projeto
├── js/
│ └── app.js # JavaScript modular da aplicação
├── assets/ # Imagens, ícones e SVG
└── README.md # Este arquivo
