Começar

O que são HTML, CSS e JavaScript?

HTML

HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar e dar significado ao conteúdo web.

<!DOCTYPE html>
<html>
<head>
    <title>Minha Página</title>
</head>
<body>
    <h1>Olá Mundo</h1>
    <p>Este é um parágrafo.</p>
</body>
</html>

Quando usar: Sempre que precisar estruturar conteúdo na web - textos, imagens, formulários, links, etc.

CSS

CSS (Cascading Style Sheets) é usado para estilizar e layoutar elementos HTML, controlando aparência e design.

body {
    font-family: 'Inter', sans-serif;
    background-color: #F3F4F6;
    color: #111827;
}

.card {
    background: white;
    border-radius: 8px;
    padding: 1.5rem;
    box-shadow: 0 4px 6px rgba(0,0,0,0.1);
}

Quando usar: Para definir cores, fontes, layouts, espaçamentos e tornar o site visualmente atraente e responsivo.

JavaScript

JavaScript é uma linguagem de programação que permite criar conteúdo dinâmico e interativo em páginas web.

// Exemplo simples de JavaScript
const button = document.querySelector('#myButton');

button.addEventListener('click', () => {
    alert('Botão clicado!');
    document.body.classList.toggle('dark-mode');
});

Quando usar: Para adicionar interatividade, manipular DOM, fazer requisições assíncronas e criar aplicações complexas.
